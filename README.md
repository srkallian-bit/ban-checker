# Ban Checker

Версия: 1.0.6 

---

## Установка

1. Скачай `ban-checker.exe` из папки `dist/` (или из Releases)
2. Двойной клик — сервер запустится фоном, браузер откроется сам
3. В настройках (шестерёнка) вставь Cookie из admin.umbrellarp.shop

## Остановка сервера
WIN + R -> cmd
`taskkill /f /im ban-checker.exe`
