# Ban Checker

Версия: 1.0.1 | Размер: 42.0 MB | Дата: 2026-05-03

---

## Установка

1. Скачай `ban-checker.exe` из папки `dist/` (или из Releases)
2. Двойной клик — сервер запустится фоном, браузер откроется сам
3. В настройках (шестерёнка) вставь Cookie из браузера

## Обновление

При запуске программа проверяет обновления на GitHub.
Если есть новая версия — появится экран с кнопкой "Download and Update".
Нажми — старый файл заменится на новый автоматически.

## Остановка

`taskkill /f /im ban-checker.exe`

## Данные

Cookie и баны хранятся в:
`C:\Users\ИМЯ\AppData\Roaming\ban-checker\data.db`
